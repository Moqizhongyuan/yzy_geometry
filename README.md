# 简介

## 这是什么

这是一款小型的图片编辑工具，目前实现矩形的平移、缩放、旋转操作，可以缩放和平移坐标系，且缩放坐标系的前提下图形不会失真

## 设计思想

该编辑器的设计思想借鉴于某位可视化领域的大佬，底层主要是事件驱动机制，通过事件发布-订阅模式来重绘制界面，原型是实现图片的变换，本编辑器将尽可能多的实现其他图形的变换操作
